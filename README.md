# Instagram-post-performance-using-PowerBI
A Power BI dashboard that analyzes Instagram post performance using key metrics such as likes, comments, saves, shares, profile visits, and follower conversion to identify top-performing content and engagement trends.
## Overview
This project presents an interactive Power BI dashboard designed to analyze Instagram post performance and audience engagement. The dashboard helps identify top-performing content by tracking key metrics such as likes, comments, saves, shares, profile visits, and follower conversions.

## Objectives
- Analyze Instagram post engagement.
- Identify the most liked posts.
- Measure audience interaction through comments and saves.
- Track profile visits and follower conversion rates.
- Discover content performance trends.

## Key Metrics
- Total Likes
- Total Comments
- Total Saves
- Total Shares
- Total Profile Visits
- Total Followers

## Dashboard Features
### Performance Overview
Displays overall Instagram engagement metrics using KPI cards.

### Top Performing Posts
Highlights the top 10 posts based on total likes.

### Engagement Analysis
Visualizes the relationship between likes, comments, and saves using a scatter chart.

### Profile Visit to Follow Conversion
Compares profile visits with follower growth to evaluate conversion effectiveness.

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- Data Visualization

## Insights
- Identified posts with the highest engagement.
- Analyzed audience interaction patterns.
- Evaluated follower conversion performance.
- Compared engagement metrics across different posts.

## Conclusion
This dashboard provides valuable insights into Instagram content performance and audience behavior, helping creators and marketers make data-driven decisions to improve engagement and reach.
